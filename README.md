# Color-Extractor
Extract color from given image

## How to Use
Click dotted area to upload image. Drag and drop image is also allowed.

To change the uploaded iamge, click the uploaded image or drag and drop over the uploaded image.

It supports almost all types of image format including jpg, jpeg, png, gif, webp, and svg.

## Features
* Upload image to extract dominant colors.
* Color picker tool to get a color from the image directly.
* Click color boxes to copy hex code of the color.
